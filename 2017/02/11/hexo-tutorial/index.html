Hexo简易教程

安装

  1. 安装nodejs,要添加到系统变量PATH中。
  2. 安装Git。
  3. 安装Hexo。
    npm install -g hexo-cli

建站

在命令行中输入下列命令：

// 初始化
hexo init <folder>
// 切换目录    
cd <folder>
// 安装依赖包
npm install

执行成功后会生成如下文件：

.
|--_config.yml  配置文件
|--package.json 应用程序信息。包含了依赖包列表
|--scaffolds    模板文件所在目录
|--source       源文件目录
|--+--_drafts   草稿目录
|--+--_posts    博文目录
|--themes       主题目录

新建博文

在命令行中输入下列命令：

hexo new <tiltle>

执行成功,会根据模板在source\_posts目录下生成date-<title>.md博文文件。

编写博文

打开博文文件进行文本编辑即可。

测试博文效果

在命令行中输入下列命令：

hexo server

执行成功，会启动一个可检测文件变动的本地服务器。一般用来测试博文效果。修改博文后，只需刷新浏览器页面http://localhost:4000/，即可看到最新状态的博文。

发布

在命令行中输入下列命令：

hexo d -g

常见问题

  1. npm安装依赖包失败。
    改用cnpm安装依赖包。执行参数不变。

  2. 安装自定义主题。
    根据主题的安装步骤执行即可。例如even主题

  3. git发布失败，认证失败。
    ssh的key需要没有密码的。详细教程参见hexo+github部署

  4. npm换源。
    详细教程参见npm换源

  5. 配置文件的配置。
    详细教程参见配置

  6. 图片放置问题
    在source文件下新建一个图片目录，将图片放置其下，通过/imagesfolder/image.png引用。详细教程参见资源文件夹
