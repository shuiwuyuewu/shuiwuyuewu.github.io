MPC-HC开发环境搭建

Feb 10, 2017

搭建的具体步骤：

  1. 安装Visual Studio 2015;
  2. 安装DirectX SDK (June 2010);
  3. 安装GCC环境，下载MSYS/MinGW包，解压到“C:\MSYS”;
  4. 下载MPC-HC代码，解压到“C:\mpc-hc”;
  5. 可选。安装GIT到”C:\Program Files\Git”目录下；不安装时，请删除”C:\mpc-hc\build.user.bat”文件中SET "MPCHC_GIT=C:\Program Files\Git"行；
  6. 可选。安装Python到”C:\Python27”目录下；不安装时，请删除”C:\mpc-hc\build.user.bat”文件中SET "MPCHC_PYTHON=C:\Python27"行；
  7. 打开”C:\mpc-hc\mpc-hc.sln”，进行编译即可。
